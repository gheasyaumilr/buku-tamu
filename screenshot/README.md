# Screenshots

Simpan screenshot UI di folder ini. Gunakan penamaan yang jelas, contoh:

- `form-mobile.png` — tampilan form pada ponsel
- `daftar-tamu-desktop.png` — tampilan daftar tamu pada desktop

Jika Anda ingin screenshot tampil di README utama, tambahkan gambar ke folder ini
dan gunakan sintaks Markdown di `README.md`:

```markdown
![Form mobile view](screenshot/form-mobile.png)
```
